jQuery DataTables oData connector

jQuery DataTables oData connector supports following oData service versions: v4, v3, and v2. oData version needs to be specified during dataTables initialization (through "iODataVersion" parameter). Also, for cross-domain requests you need to set "bUseODataViaJSONP" to true.
